# Vansha Database Architecture

## Core Tables
### users
Authenticated platform accounts.

### persons
Canonical human identity records.

### user_person_links
Maps accounts to person records.

### family_spaces
Collaborative family contexts.

### space_memberships
Membership and role mapping.

### relationships
Graph edges between persons.

### relationship_types
Relationship taxonomy.

### invitations
Invite and claim workflow.

### claims
Profile ownership claims.

### privacy_rules
Visibility governance.

### branch_access_requests
Permission-based traversal requests.

### merge_requests
Duplicate resolution workflows.

### audit_logs
Change history.

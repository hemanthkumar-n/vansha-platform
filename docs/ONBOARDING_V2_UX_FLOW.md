# Vansha Onboarding V2 UX Flow

## Step 1: Identity
Capture:
- Legal Name
- Display Name
- Preferred Name
- Gender
- DOB
- Phone
- Email
- Country
- State
- Native Place
- Mother Tongue

## Step 2: Naming Intelligence
Ask:
- Do you use a family surname?
- Is father's name part of your naming convention?
- Any cultural naming conventions?

Examples:
- Tamil naming
- Telugu surname naming
- Kerala variations

## Step 3: Create Family Space
Examples:
- Natakam Family
- Doguparti Family
- Friends Circle

## Step 4: Add Immediate Family
Quick add:
- Father
- Mother
- Spouse
- Children
- Siblings

## Step 5: Relationship Intelligence
When user selects a basic relationship like Uncle:

Ask:
"How do you call this person?"

Dropdown examples:
- Uncle
- Mama
- Mamaji
- Mamu
- Babai
- Mavayya
- Chithappa
- Periyappa
- Machan
- Chettan
- Anna
- Other

If Other:
- Custom relationship term input

Additional optional prompts:
- Is this commonly used in your region?
- Family-specific nickname?
- Language?
- State / region?

Examples:
- Papa = pan India father
- Achan = Malayalam father
- Amma = multi-language mother

## UX Rule
Canonical relationship != display term.
